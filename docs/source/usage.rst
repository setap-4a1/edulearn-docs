Installation
------------

To run Edulearn, clone the repo at https://github.com/setap-4a1/Edulearn, install dependencies, and run the Flask project:

.. code-block:: console

   $ git clone https://github.com/setap-4a1/Edulearn.git
   $ cd Edulearn
   $ pip install -r requirements.txt
   $ flask run
